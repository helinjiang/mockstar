快速开始，只需两步

只需一个命令即可初始化一个 Mock Server 项目，一个命令启动项目。已提供多种场景支持，拿来即用。


初始化一个新项目，例如 my-app, 可以运行如下命令:

npx create-mockstar-app my-app

启动项目, 可以运行如下命令:

npx mockstar start --watch

不只是 Mock Server，更是 Mock 数据管理专家

提供配套的 UI 后台，可视化管理你的所有 Mock 数据，并可任意指定返回所需要的 Mock 数据上。

纯 JavaScript，轻松掌握无门槛

无需准备额外的技能，使用 JS 语言来写 mock 数据，只需满足 CommonJS 规范，同步/异步皆支持，pure object、function、json 等形式任君选择。
